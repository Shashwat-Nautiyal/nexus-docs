# Avail Nexus SDK reference

## Setting up the dev environment

<Callout type="info">
**BEFORE WE START**

1. The Nexus SDK is designed for browser environments and requires an injected wallet provider (e.g. MetaMask, WalletConnect, etc.).
2. The SDK cannot be used in any server-side environments.
</Callout>

## Setting up the dev environment

1. Install the Nexus SDK using your package manager of choice:

<Tabs items={['npm', 'yarn', 'pnpm']}>

<Tabs.Tab>
```bash showLineNumbers filename="Terminal"
npm install avail-nexus-sdk@0.0.3-dev.3
```
</Tabs.Tab>


<Tabs.Tab>
```bash showLineNumbers filename="Terminal"
yarn add avail-nexus-sdk@0.0.3-dev.3
```
</Tabs.Tab>


<Tabs.Tab>
```bash showLineNumbers filename="Terminal"
pnpm add avail-nexus-sdk@0.0.3-dev.3
```

</Tabs.Tab>
</Tabs>

2. Set up a frontend project in your preferred manner.
Any stack frontend works, as long as it can provide an injected wallet provider to the SDK.
