# `transfer(params)`

Use the `transfer` function to send funds from the unified balance to any wallet (EOA) on any chain using approved source chain funds.

## Parameters

```typescript showLineNumbers filename="Typescript"
interface TransferParams {
  token: SUPPORTED_TOKENS;
  amount: number | string;
  chainId: SUPPORTED_CHAINS_IDS;
  recipient: `0x${string}`;
}
```

## Transfer Example

<Callout type="warning">
Please go through the [overview page](/api-reference/avail-nexus-api/overview) to understand how to set up the SDK and the required hooks before using any SDK function that moves around user funds.
</Callout>

Here are some minimal examples of how to use the `transfer` function:

```typescript showLineNumbers filename="Typescript"
// Transfer to specific recipient
await sdk.transfer({
  token: 'ETH',
  amount: 1,
  chainId: 1, // on Ethereum
  recipient: '0x742d35Cc6634C0532925a3b8D4C9db96c4b4Db45',
});
 
// Transfer USDC on Polygon
await sdk.transfer({
  token: 'USDC',
  amount: '250.75',
  chainId: 137,
  recipient: '0xRecipientAddress',
});
```
