# Fetch Header by Avail Block Hash

Use this endpoint to get the Nexus header information for a specific Avail block using it's hash.

### Params

- `hash`: Avail block hash (hex-encoded string)

### Response

- Nexus header information, including:

    **Still in progress**

### Example Request

<Callout type="info">
The `hex-encoded` params must be supplied without the `0x` prefix.
</Callout>

```bash filename="CURL"
curl "https://dev.nexus.avail.tools/header?hash=010133f52baede81baf3408ec4be909ac619f2ed98902ffbe318990716ae7b5a"
```
